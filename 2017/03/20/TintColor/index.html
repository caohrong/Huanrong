Tint Color

苹果在iOS中添加了Tint Color属性，看似不起眼的tintColor属性其实相当的强大，本章介绍怎样在controls里面使用tintColor以及怎么样让他来渲染图片。

现有iOS控件的Tint Color
在iOS7中UIView添加了新的TintColor属性，由改变UIView子类的用来更改应用程序的外观。tintColor默认是没有值，但是他会从继承父视图的tintColor参数值，如果所有的父视图都没有tintColor属性，那么他会被默认设置为蓝色。所以你可以设置root view controller的TintColor属性，从而影响到整个应用的外观。

tintAdjustmentMode
除了能够设置tint Color，UIView还有另外一个属性允许你调暗tint color的颜色。因此他也会调整整个视图结构，这个属性有三种值：UIViewTintAdjustmentModeNormal, UIViewTintAdjustmentModeDimmed，UIViewTintAdjustmentModeAuto.

当tinColor被改变的时候，被系统调用。系统将会调用所有子视图中的tintColorDidChange的方法。所以需要在实现改方法中重绘视图。
