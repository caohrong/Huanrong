Navigation Bars

导航栏可以让您有组织且直观的方式展示应用的内容，navigation bar被显示在屏幕的顶部，并且包含用于浏览屏幕层次的按钮。一个navigation bar 一般包含一个返回按钮，一个标题，和一个右按钮。经常与navigation controller一起使用，当然你可以在你的app里面单独使用。

###作用：

  • 返回到之前的视图
  • 切换到新的视图

###Appearance of Navigation Bars

##Navigation Bars的相关属性

Bar Style
Navigation bars 有两种样式，
UIBarStyleDefault：白色透明背景

UIBarStyleBlack

其中UIBarStyleBlackOpaque 和 UIBarStyleBlackTranslucent已经作废。

##Tint Color
你可以指定自定义的tint color 作为navigation bar的背景颜色，默认的背景颜色是白色。

另外，你可以设置自定义的tint color，设定的Tint color 将会影响到导航栏的按钮图片和标题的颜色，如果没有设定自定义的tint color那么navigation bar会继承父视图的tint color，如果父视图也没有设置tint color那么默认会使用蓝色。

##Images
你可以为navigation bar自定义背景图片，你可以使用setBackgroundImage:forBarMetrics:.方法，一定要指定metrics的参数，因为在不同的设备上和不同的设备方向上（横屏以及竖屏），navigation bar有不同的尺寸。

可以通过shadowImage设置navigation bar的自定义阴影图片，一旦你设置了自定义的阴影图片你必须同时也设定背景图片。

##Translucency
iOS7中导航栏默认都是透明的，另外，所有的navigation bars上都有系统模糊效果。他允许你的内容透过navigation bar模糊显示。

不管是设置了barStyle样式，还是设置了barTintColor的自定义颜色，Translucency默认都是打开的。但是你可以通过将transluency参数设置为no而达到将navigation bar设置为不透明的背景。在这种情况下，无论是UIBarStyleBlack还是UIBarStyleDefault或者是设置了barTintColor都会是不透明的背景。

  • 如果navigation bar有一个自定义的背景图片，透明度将会从图片的alpha的平均值推算出来。如果平均的alpha值不超过1.0，那么navigation bar默认就会是透明的，如果平均值是1.0那么naivgation bar 默认就是不透明的。
  • 如果你在背景图片不透明的的navigation bar上把translucency设置为true，那么navigation bar会使这背景图片变得透明。
  • 如果你在背景图片透明的navigation bar上把translucency设置为false，于此同时你的Bar style样式为UIBarStyleBlack那么将会为透明图片提供黑色的背景从而达到不透明的效果。
  • 如果你的Bar style样式为UIBarStyleDefault，那么会提供白色的背景颜色。
    *如果你设置了barTintColor那么就会提供你设置的barTintColor为背景颜色。

##Title Attributes
可以通过titleTextAttributes参数指定bar’s title文本的字体，颜色，文字阴影颜色，阴影的偏移值。
setTitleVerticalPositionAdjustment:forBarMetrics: 你可以通过该方法指定不同场景的bar’s title的颜色，同样也要指定metrics参数，一定要指定metrics的参数，因为在不同的设备上和不同的设备方向上（横屏以及竖屏），navigation bar有不同的尺寸。
