Basic

IBInspectable

在变量类型前加IBInspectable，那么可以在xib中直接通过UserDefined Runtime Attributes给该变量设置属性。

IB_DESIGNABLE
在自定义的类前面加上IB_DESIGNABLE并配合IBInspectable可以实现在xib中修改的参数直接得到显示结果。
