关于集合

Cocoa Touch中有array、dictionary和set集合类型

所有的集合都有几下共同的特性，多数集合都只允许存放object对象并且都有可修改和不可修改两种类型(NSArray 和 NSMutableArray）

共同的特性：

* 枚举集合中的对象
* 验证对象是否存在
* 获取集合中单个对象

可以修改的集合共同的特性：

* 添加对象到集合中
* 删除集合中的对象
集合类型 集合名词 集合特点
Array NSArray 和 NSMutableArray 1. 有序集合 2. 允许通过索引访问元素
Dictionaries NSDictionary and NSMutableDictionary 1. 无序结合 2. 允许通过键值访问元素 3. 允许快速的插入和删除操作 4. 通过键值排序的时候非常有用
Sets NSSet, NSMutableSet, and NSCountedSet 1. 无序的集合 2. 允许快速的插入和删除操作 3. 快速验证集合中是否存在某个对象 4. NSSet, NSMutableSet无法存储相同的对象 5. NSCountedSet允许存储不同的对象

  • Arrays排序的子类：Index Sets（NSIndexSet and NSMutableIndexSet）
    Index Sets是为了扩展Array而存在的子类，他们允许你储存分组通过添加indexes到数组中而不是重新创建一个数组，在开发中我们可以用它来存储用户多选时记录用户选中的row和section信息。

Array

Array中包含的对象可以是不同类型的对象

Array基础

NSArray对象是不可修改的对象，也就是说array一旦被创建以后你不能添加、删除、或者替换对象，但是你可以单独修改里面对象，如果改对象是可以修改的话。

NSMutableArray是可以被修改的array，允许添加或者删除，当你需要的时候分配新的内存。如果数组产生了渐变或者非常大的变化，那么应该使用可变数组，因为大结合需要更多的时间进行初始化。

初始化

注意：Most operations on an array take constant time: accessing an element, adding or removing an element at either end, and replacing an element. Inserting an element into the middle of an array takes linear time.
