祈使句

Veröffentlicht am 2017-06-13 | in English

祈使句就是把主语You 以及助动词 should 或 must 省略的句子。词类句型一开始即为原型动词。句尾可视语气强弱置感叹号或句号。

  • You must run now => Run！=> Don’t run!
  • You must be quiet now => Be quiet! => Don’t be quiet!
  • You must send hime a letter => Send him a letter => Don’t send hime any letters.

否定句的祈使句有两种：一为Don‘t 起首，另一为Never（绝不）起首。
