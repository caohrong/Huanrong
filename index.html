enumerations

An enumeration is a type that can hold a set of integer values specified by the user.
C++中包含两种枚举类型：
Plain enums:枚举值与枚举类型本身在用一个作用域中，枚举值会隐形的转为整数。
enum class: 他们枚举的值作用域在枚举的内部，枚举值不会隐式的转为其他的类型。

POSIX简介

Posted on 2019-06-06 | In POSIX |

最近发现许多GitHub的源码，引用都来自于iOS中/usr/includ目录的头文件，后面查到这些头文件其实都是基于POSIX的标准。查了一下POSIX是由IEEE制定的标准，在认证的系统里面，基于POSIX开发的软件，具有移植性。而常用的macOS、iOS、Android以及Linux系统，基本上都是和POSIX兼容的。最典型的例子就有Tencent开源的网络框架Mars，就是基于POSIX实现的可移植性。
